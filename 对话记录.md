# 定积分计算器 — 开发对话记录

## 日期
2026-04-25

## 项目概览

本次会话从头搭建了一个基于命令行的定积分计算器 C++ 工程，包含 CLI 和 GUI 两个版本。

## 完成的工作

### 1. 项目结构初始化
- 创建了 `src/`、`include/`、`external/exprtk/`、`build/` 目录
- 引入 **exprtk**（header-only 数学表达式解析库，~1.6 MB）作为唯一外部依赖
- 构建系统：**Makefile**（MinGW g++）和 **CMakeLists.txt**（VS 2022）

### 2. 数值积分核心 (`include/integrator.h` + `src/integrator.cpp`)
实现了三种数值积分方法：
- **梯形法 (Trapezoidal)** — O(n) 次求值，精度 O(1/n²)
- **Simpson 1/3 法则** — 要求偶数分段，精度 O(1/n⁴)
- **自适应 Simpson (Adaptive Simpson)** — 递归细分，误差控制

所有方法自动处理 a > b 和 a == b 的边界情况。

### 3. 表达式解析封装 (`include/parser.h`)
对 exprtk 做了简洁封装，支持 `x` 作为自变量，内置了全部数学常数和函数。

### 4. CLI 版本 (`src/main.cpp`)
```
integral <expression> <lower> <upper> [options]
```
参数：
- `--method <trapezoidal|simpson|adaptive>` — 选择方法
- `--n <int>` — 梯形法/辛普森法的分段数
- `--tol <float>` — 自适应方法的误差容限
- `--help` — 帮助信息
- `--methods` — 列出支持的数学函数

### 5. GUI 版本 (`src/gui.cpp`)
Windows 原生图形界面（Win32 API），使用 **Unicode API** 确保中文正确显示。

包含：
- 输入栏：被积函数、积分下限、积分上限、精度控制
- 下拉选择：计算方法（自适应 Simpson / 辛普森 / 梯形法）
- "计算定积分" 按钮
- 结果显示（22pt 粗体）+ 求值次数 + 方法信息

**解决中文乱码问题**：将所有 `CreateWindowExA` 等 ANSI API 替换为 `CreateWindowExW` 等 Unicode API，并添加 `WideToUTF8` / `UTF8ToWide` 转换函数。

## 构建命令

| 命令 | 说明 |
|---|---|
| `mingw32-make all` | 编译 CLI + GUI 两个版本 |
| `mingw32-make integral` | 仅编译 CLI |
| `mingw32-make integral_gui` | 仅编译 GUI |
| `mingw32-make run` | 运行 CLI |
| `mingw32-make run-gui` | 运行 GUI |
| `mingw32-make clean` | 清理构建产物 |
